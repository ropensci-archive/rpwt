**A general purpose R interface to the [Pensoft Writing Tool](http://pwt.pensoft.net/)**

## Quick start

### Install

```r
install.packages("devtools")
devtools::install_github("ropensci/rpwt")
library('rpwt')
```

### Do stuff

```r
asdfadf
```
